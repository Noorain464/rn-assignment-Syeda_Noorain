A minimal **React Native** app to write and preview **Markdown notes**. Built with **Expo Router** and styled using **NativeWind**.

## Features

- Edit notes in **Markdown** (Bold, Italic, Lists, Headers).  
- Toggle between **Editor** and **Preview** modes.  
- **Word count** updates in real-time.  
- **Clear** notes with confirmation.
